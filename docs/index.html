Yubidrive

Store files securely.

Use your Yubikey to encrypt and securely store files. Download and decrypt from any device as long as you have your Yubikey with you.
Secure
Files are encrypted on the device using your Yubikey. Unencrypted files will never leave your device.
Device agnostic
As long as you use the same Yubikey, it doesn't matter which device you use to manage your files.
Anonymous
No user account needed.
